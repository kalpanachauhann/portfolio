# Kalpana Chauahan - Developer Portfolio

A personal portfolio website showcasing my projects, skills, and background.

## Features
- Responsive single-page design (mobile + desktop)
- Typing animation for roles
- Smooth scroll navigation with active state
- Project cards with GitHub links
- Education timeline
- Dark theme with cyan/purple accents

## Tech Stack
- HTML5
- CSS (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript

## Sections
- Hero (intro + typing animation)
- About Me
- Skills
- Projects
- Education
- Contact

## How to Run
Just open index.html in any browser.
Or run with Python:
    python -m http.server 8000
Then open: https;//localhost:8000
